# cardgame-java
Dit is een eenvoudig hoger/lager kaarstpel in Java.

Er wordt een willekeurige kaart getoond (waarde + kleur).
De speler gokt of de volgende kaart hoger of lager zal zijn. 
Zolang je juist gokt en kiest om verder te spelen, gaat het spel door.

Belangrijke concepten vanuit vakken: 

-Enumeraties (H5 Programming Advanced)
-Class (H3 Programming Advanced)
-Op een veilig manier gebruikersinput vragen (Programming Fundamentals 1)
-Lusstructuren (Programming Fundamentals 1)

Dit project is een challenge uit de cursus Progamming Advanced 
(Module 5 - Hard Challenge)
